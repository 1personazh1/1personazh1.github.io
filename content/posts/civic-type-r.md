# Honda Civic Type R: Король переднеприводных хот-хэтчей

![Honda Civic Type R](civic-type-r.jpg "Honda Civic Type R")

Honda Civic Type R — это воплощение философии **Type R**: максимальная производительность при минимальном весе. Никакого полного привода, никаких турбин (до 2015 года), только чистая мощность и VTEC.

## Двигатели VTEC — душа Type R

Сердце любого Civic Type R — это высокооборотистый атмосферный двигатель с системой изменения фаз газораспределения **VTEC**.

## Что такое VTEC?

**VTEC** (Variable Valve Timing and Lift Electronic Control) — система, которая меняет профиль кулачков распределительного вала на высоких оборотах.

Когда стрелка тахометра переваливает за 5500-6000 об/мин, происходит знаменитый **VTEC kick** — двигатель резко меняет характер, мощность растет, и звук становится по-настоящему гоночным.

## Тюнинг Civic Type R

Благодаря надежности двигателей K-series, Type R отлично тюнингуется:

- **Чип-тюнинг** — +30-50 л.с.
- **Интеркулер** — лучшее охлаждение наддува
- **Выхлопная система** — снижение веса и улучшение звука
- **Подвеска** — койловеры для трека

## Заключение

Honda Civic Type R — это автомобиль для тех, кто ценит драйв, высокие обороты и японскую надежность. Это последний настоящий переднеприводный хот-хэтч с душой гонщика.
